# monsters-rolodx
View Demo: https://adityaraval.github.io/monsters-rolodx/.

Features:
- Image Gallery with Search and filter


Steps:

For Development:

```
yarn
yarn start
```

For Production Build

```
npm run build
```

and open `dist/index.html` in browser.

For Deployment
```
npm run deploy
```
